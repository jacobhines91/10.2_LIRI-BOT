# 10.2_LIRI-BOT
Liri is a Language Interpretation and Recognition Interface. Use the commands below  in the command line to interact with the node app
## LIRI commands

concert-this
```
spotify-this-song
```
movie-this
```
do-what-it-says
```

## Technologies used
* Node.js
* Javascript
